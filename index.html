3 En línea (^(x.0)^)

¿Qué es?

El juego de 3 en línea, también conocido como juego del gato, es un pasatiempo clásico y sencillo que ha entretenido a personas de todas las edades.

¿En qué consiste?

Tablero: Se utiliza un tablero de 3x3 casillas, que forma una cuadrícula.

Jugadores: Participan dos jugadores, cada uno con un símbolo diferente, generalmente una "X" y una "O".

Objetivo: El objetivo de cada jugador es colocar tres de sus símbolos en línea recta, ya sea horizontal, vertical o diagonalmente, antes que su oponente.

Turnos: Los jugadores se turnan para marcar una casilla vacía con su símbolo.

Fin del juego: El juego termina cuando un jugador logra formar una línea de tres símbolos o cuando todas las casillas están ocupadas y ningún jugador ha ganado (empate).

Implementación

Para la implementacion de nuestro juego decidimos usar dos clases, Juego y Boton.

Clase Juego

str Tablero : Atributo de instancia. Matriz que representa el estado del tablero

str Turno : Atributo de instancia. Indica de quien es el turno actual (x o 0)

str Botones : Atributo de instancia. Almacena los botones que se colocan en el tablero

tk Master : Todos los elementos que se crean dentro de la clase juego (como los botones en este caso) se asocian a la ventana principal mediante el argumento master.

Void __init__ : Inicializa el juego

Void Empezar_el_juego (self, int fila, int columna): Genera las filas y columnas dentro del tablero, asigna el turno X o O.

Void Verificar_ganador (self) : Comprueba si hay una fila, columna o diagonal con el mismo simbolo (X o O)

Void Reiniciar (self): Elimina el texto que contiene los botones

Clase Boton

  • Atributos

int Fila: Del boton del tablero

int Colunma: Del boton del tablero

juego: Instancia de la clase juego.

  • Metodos

Void __init__ (self, tk master, int fila, int columna, instancia juego): Inicializa el boton

Void Actualizar_estado (self, str texto): Actualiza el texto del botón ya que por defecto esta vacio, y hace visible una X o un O
